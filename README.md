# Sleeping

一个纯前端的睡眠管理小应用，用来快速记录睡眠、醒来、没睡着起身，并在本地生成睡眠统计和可视化分析。

## 功能

- 一键记录：`我醒了`、`开始睡了`、`没睡着，起来了`
- 自动处理“开始睡了但没睡着”的情况
- 支持补录清醒或睡眠时间段
- 支持文字补录，例如：`2:10-2:40 醒着`
- 统计今日睡眠时长、清醒次数、零散程度
- 智能分析 7 天平均、睡眠债、规律性和睡眠评分
- 可切换一星期 / 一个月趋势图
- 可切换睡眠、醒来、零散、评分等可视化指标
- 7 天 24 小时睡眠热力图
- 可设置个人目标睡眠时长
- 可做晨间复盘：睡眠质量、醒后精神、影响因素和备注
- 支持导出 / 导入本地睡眠数据
- 内置银河紫蓝视觉和睡眠相关 SVG 图标
- 支持添加到手机主屏幕，带自定义 App 图标

## 使用方式

直接用浏览器打开：

```text
index.html
```

数据保存在当前浏览器的 `localStorage` 中，不需要后端服务。

## 添加到手机主屏幕

部署到 GitHub Pages 后，用手机浏览器打开页面：

- iPhone Safari：点分享按钮，选择 `添加到主屏幕`
- Android Chrome：点菜单，选择 `添加到主屏幕` 或 `安装应用`

应用会使用仓库里的 `icon.svg` 和 `manifest.webmanifest` 作为主屏幕图标和安装信息。

## 智能功能

- 睡眠债会按你设置的目标睡眠时长计算，默认目标是 8 小时。
- 晨间复盘会保存每天的主观状态，后续可以对照睡眠评分、醒来次数和零散程度。
- 影响因素标签包括咖啡因、屏幕、压力、运动和午睡。
- 导出的 JSON 包含睡眠事件、每日复盘和个人设置。

## 记录逻辑

- 点 `开始睡了`：记录一段睡眠开始。
- 点 `我醒了`：结束当前睡眠并记录醒来。
- 如果点 `开始睡了` 后 45 分钟内又点 `我醒了`，应用会自动判断为没睡着，不把这段计入睡眠。
- 如果躺了更久但确认没睡着，可以点 `没睡着，起来了`，应用会撤销上一次睡眠开始。

## GitHub Pages 部署

如果仓库名是 `sleeping`，把这些文件放在仓库根目录：

```text
index.html
styles.css
app.js
icon.svg
manifest.webmanifest
README.md
```

然后在 GitHub 仓库中：

1. 打开 `Settings`
2. 进入 `Pages`
3. `Source` 选择 `Deploy from a branch`
4. `Branch` 选择 `main`
5. 目录选择 `/root`
6. 保存

部署完成后访问：

```text
https://你的GitHub用户名.github.io/sleeping/
```

## 隐私

当前版本不上传任何数据。所有记录只保存在你使用的浏览器本地。清除浏览器数据或换设备后，记录不会自动同步。
